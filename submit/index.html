Game Submission Request

Fill out our form to request a new game. Our team will review your submission and add the game if it meets our criteria.

What We're Looking For

  • Games with an active challenge run community
  • Clear definitions for challenge types (hitless, damageless, etc.)
  • Established run categories
  • At least a few documented runs
