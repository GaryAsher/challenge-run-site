How to install mods for Hollow Knight:

  1. Install the Modding API

    Download and install the Hollow Knight Modding API. This is required for all mods.

  2. Download the Gun Mod

    Get the latest version of HKGun/Hollow Point from the link above.

  3. Install to Mods Folder

    Place the mod .dll file in your Hollow Knight/Mods folder.

  4. Configure Your Settings

    Launch the game and configure gun settings in the mod menu.

  5. Verify Installation

    Make sure the gun appears in-game before starting a challenge run.
