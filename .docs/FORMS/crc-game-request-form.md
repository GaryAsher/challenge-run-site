# CRC Game Request Form Specification

## Form Title
**Request a Game for CRC**

## Form Description
Help us grow the Challenge Run Community! Tell us about a game you'd like to see tracked on CRC. We'll review submissions and prioritize based on community interest.

---

## Questions

### 1. Game Name (Required)
**Type:** Short answer  
**Placeholder:** "e.g., Hollow Knight, Elden Ring, Celeste"

---

### 2. Platform(s)
**Type:** Checkboxes  
**Options:**
- PC
- PlayStation
- Xbox
- Nintendo Switch
- Mobile
- Other

---

### 3. What challenge types would you like to see tracked?
**Type:** Checkboxes  
**Options:**
- Hitless / No Hit
- Damageless
- No Death
- Speedrun Categories
- Restricted Runs (limited items/abilities)
- Other (please specify)

**Helper text:** "Select all that apply"

---

### 4. Are there existing communities for this game's challenge runs?
**Type:** Multiple choice  
**Options:**
- Yes, there's an active community
- Yes, but it's small
- Not that I know of
- I'm not sure

---

### 5. Links to communities (Optional)
**Type:** Long answer  
**Placeholder:** "Discord servers, subreddits, YouTube channels, etc."

---

### 6. Why should this game be added?
**Type:** Long answer (Optional)  
**Placeholder:** "Tell us what makes this game great for challenge runs"  
**Max length:** 500 characters

---

### 7. Your Discord username (Optional)
**Type:** Short answer  
**Placeholder:** "e.g., username#1234 or @username"  
**Helper text:** "So we can follow up if we have questions"

---

### 8. Email (Optional)
**Type:** Short answer  
**Validation:** Email format  
**Helper text:** "For updates on your request (we won't spam)"

---

## Form Settings

- **Collect email addresses:** Optional
- **Limit to 1 response:** No (allow multiple game requests)
- **Edit after submit:** Yes
- **Confirmation message:** "Thanks for your submission! We review requests regularly and prioritize based on community interest. Join our Discord to stay updated on new game additions."

---

## Notes for Form Creation

1. Keep required fields minimal (only Game Name)
2. Use checkboxes for multi-select options
3. Add "Other" options where appropriate
4. Keep descriptions brief and friendly
5. Mobile-friendly by default (Google Forms handles this)
